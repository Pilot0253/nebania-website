About Nebula OS

Nebula OS is an operating system developed and maintained by Pilot0253 and contributors from the Nebula OS Discord community. It is optimized for Raspberry Pi 5 devices and focuses on lightweight performance, security, and flexibility for enthusiasts and developers.

Link-Lang

Link-Lang is the official programming language for Nebula OS, designed to be simple and efficient for system-level development. It is maintained by the Nebula OS community and developed by lowisfigo17.

Official Link-Lang Repository

Kotel

Kotel is an experimental package manager developed within the Nebula OS community. It allows users to install, update, and manage software packages across multiple systems.
